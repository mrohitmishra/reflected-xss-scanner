# Reflected XSS Scanner

A small, simple Python-based reflected XSS scanner.  
It injects payloads into URL parameters and checks if the server reflects them back, helping identify possible reflected XSS points.



---

## Features

- Supports **GET** and **POST** requests  
- Simple reflection detection (substring match)  
- Payloads generated by a **PayloadGenerator** class  
- Covers at least **three contexts**, including **attribute-name**  
- Generates either:
  - Terminal output, or  
  - HTML report file (optional)
- Easy to modify or extend for more contexts and payloads

---

## Installation

pip install -r requirements.txt

## Usage

Basic scan:
python scanner.py --url "https://target.com/page.php" -p q --method GET

## CLI Options
--url        Target URL  
--method     GET or POST  
--params     Comma-separated parameters (example: q=hello,id=1)  
--html       Output HTML report file  
--timeout    Optional request timeout  
--help       Show help
